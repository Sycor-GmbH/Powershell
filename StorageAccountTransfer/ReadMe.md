Pre Requirements (manually Transfer Storage Account Data cross tenants)
  --> Will be Automated

1.Install AzureCLI
https://github.com/Sycor-GmbH/Powershell/blob/main/StorageAccountTransfer/AzureCLI.ps1
2. Source and Destination Tenant logins 
3. Structure of Stroage accounts containers, tables, file shares and queues should be same in both Target and Destination 
4. Then perform AZcopy and then provide access to desired resources (rbac)

Script Syntax:
azcopy copy “[source_path_of_file_url]/[SAS]" "[dest_path_of_file_url]/[SAS]" --recursive
